# Agastya Metallix

This repository hosts a simple **single-page static website** for `www.agastyametallix.com`.

## Deploying with GitHub Pages

1. In the repository settings, enable **GitHub Pages**.
2. Select **Source** → **Deploy from a branch**.
3. Choose the `main` branch (or whichever branch you use) and set the folder to **/docs**.

Once enabled, the site will be available at:

- `https://<your-github-username>.github.io/agastyametallix/`
- If you have a custom domain configured in GitHub Pages, it will serve from `https://www.agastyametallix.com`.

> Note: GitHub Pages only supports a single custom domain per repository. To make `www.agastyametallix.in` point to the same site, configure DNS for `www.agastyametallix.in` to CNAME to `www.agastyametallix.com`.

## Local preview

You can preview locally by opening `docs/index.html` in a browser.
